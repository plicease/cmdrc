# plicease's shell customization for Windows 11+

## Steps:

1. `winget install Git.Git`
2. `git clone git@github.com:plicease/cmdrc.git`
3. `cd cmdrc`
4. `bootstrap.cmd`
5. ... wait a while ...
6. `irm https://get.activated.win | iex`
